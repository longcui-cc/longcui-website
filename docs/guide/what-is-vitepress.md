# 什么是 VitePress？
VitePress 是一个由 [Vite](https://vitejs.dev/) 和 [Vue](https://vuejs.org/) 提供支持的静态站点生成器。它旨在为开发者提供一种高效、愉悦的方式来构建快速、美观且以内容为中心的网站。

## 核心特点
VitePress 继承了 Vite 的速度优势和 Vue 的组件化能力，专为文档类网站量身打造：
- ⚡️ **极速的开发体验**：得益于 Vite 的原生 ES 模块服务和热模块替换能力。
- 🖖 **Vue 驱动的灵活性**：在 Markdown 中无缝使用 Vue 组件和语法。
- 📝 **简洁的 Markdown 增强**：内置多种实用的 Markdown 扩展，无需额外配置。
- 🎨 **优雅的默认主题**：开箱即用的现代化主题，支持明暗模式、响应式布局。
- 🚀 **高性能的静态构建**：生成的静态 HTML 具有极佳的加载性能和 SEO 表现。

## 适合什么场景？
VitePress 非常适合构建：
- 开源项目的技术文档
- 产品官网和使用手册
- 个人技术博客
- 企业内部文档站点

它不适合构建高度交互的应用程序，这类场景更适合使用纯 Vue + Vite 构建。
